## Project Overview

This project is a front-end implementation of a social media platform called Twizzle, built using only HTML, CSS (with Flexbox and Grid for layouts), and Vanilla JavaScript. All data is stored locally in the browser's localStorage, providing a persistent experience without the need for a backend server. The application includes user authentication, posting, interaction features like liking and commenting, user exploration, and profile management.

## How to Use the App

1. **Open the Application**: Open `index.html` in a web browser to start the app.

2. **Sign In or Sign Up**:
   - Use the demo account: Email: `dana@example.com`, Password: `password123`
   - Or create a new account by switching to the "Sign Up" tab and filling in the required fields.

3. **Test Posting**:
   - After logging in, go to the "Home" tab.
   - Type a message in the text area (up to 280 characters) and click "Post".
   - Your post should appear in the feed.

4. **Test Interactions**:
   - Like posts by clicking the heart icon.
   - Comment on posts by clicking the comment icon and adding a reply.

5. **Explore Users**:
   - Click on the "Explore" tab.
   - Search for users by name or bio using the search bar.
   - Click on a user to view their profile and follow/unfollow them.

6. **View Profile**:
   - Click on the "Profile" tab to see your own profile.
   - View your posts, followers, and following counts.

7. **Suggested Users**:
   - Check the right sidebar (on desktop) for suggested users to follow.

8. **Logout**:
   - Click the "Logout" button in the sidebar to sign out.

All data is stored locally, so changes persist across browser sessions.
